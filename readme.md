# Protocol

## 공통 
+ **Databits** 8
+ **Baudrate** 19200
+ **Stop Bits** 1
+ **Parity** N

## 싸인보드
| 0(Start) | 1(Device Type) | 2(Power)  |  3-2bit  |  3-2bit    |  3-4bit    |  4(순차 동시) | 
|   :---:  |     :---:      | :---:     |   :---:  |   :---:    |    :---:    |  :---:      |
|   0xaa   |     0x01       | 0x01(켜짐) | 0x01(빠름)| 0x01(주간)   |  0x01(좌측)  | 0x01(동시)
|          |                | 0x00(꺼짐) | 0x02(중간)| 0x02(흐린날) |  0x02(양방향) | 0x02(순차)
|          |                |           | 0x03(느림)| 0x03(야간)   |  0x04(우측)  |
|          |                |           |          |            |   0x08(엑스)  |
